HTWK-App Project

In scope of my studies as a computer scientist I've started the HTWK-App as an open-source project. It fast evolved to an outstanding project, developed by five people. The HTWK-App is a website shaped mobile application that is published as a WebApp. It provides different services from the Leipzig University of Applied Sciences, like canteen offers, student grades, a list of employees and much more.

I've worked at the mobile application and used HTML 5, CSS 3 and Javascript. As frameworks, we decided to use JQuery, JQuery Mobile and Apache Cordova. In addition I've become the the project leader and temporarily managed up to four developers. Currently, I keep working at this project in the position just mentioned.

  • Developed for: Leipzig University of Applied Sciences
  • Period: October 2013 until today
  • Website: HTWK-App

Linux - A Developers Operating System

Since 2010 I'm using Linux. I've started with Ubuntu as a system to program C, but'm completely switched fast. Especially the surfaces of Linux got concise advantages (virtual desktops, comfortable consoles and much more) over the Windows surface. In the end of 2012, I've switched to Fedora. They got a better open-source philosophy and their repositories are way more up to date than the ones from Ubuntu. Fedora is my operating system for daily usage, regardless of multimedia or developer system.

Through the many years of Linux usage I'm familiar with the systems of Ubuntu and Fedora, representing THE two major distributions. I've dealt with the various surfaces, package managers, Linux subsystems and many other units.
